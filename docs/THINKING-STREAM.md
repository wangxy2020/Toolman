# 思考过程流式展示

## 何时可见

Toolman 会在模型返回 `reasoning_content` / `reasoning` 字段时，在消息流中展示 **「思考过程」** 折叠块。

## 使用条件

1. **推理模型**：如 DeepSeek-R1、QwQ 等会在 API 流中输出 reasoning 的模型；普通对话模型（如 `gemma4:26b`）通常无思考块。
2. **纯对话或工具结束后的最终回复**：工具多轮中间阶段使用非流式 `chatComplete`，思考内容主要在最终流式回复阶段出现。
3. **消息设置**：关闭「自动折叠思考过程」可默认展开查看。

## 故障排查

- 看不到思考块 → 确认模型是否支持 reasoning 输出
- 有内容但折叠 → 点击「思考过程」标题展开
- 使用 Claude + 工具 → 工具回合与 OpenAI 兼容路径行为一致；最终回复会流式输出
